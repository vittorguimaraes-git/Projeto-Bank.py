# 🏦 Projeto BANK

Um projeto simples em Python para demonstrar **lógica de programação** com menus, login, cadastro de usuários e fluxo de interação.  
Este projeto é **didático** e está em evolução — novas funcionalidades serão adicionadas futuramente (como saldo e transferências).

---

## 🚀 Funcionalidades atuais
- Cadastro de usuários (username e senha)
- Login e logout
- Menus interativos com opções
- Tratamento de erros em entradas inválidas (`try/except`)
- Mensagens padronizadas para melhor visualização

---

## 📌 Funcionalidades futuras
- Armazenamento de saldo por usuário
- Visualização de saldo
- Transferência de valores entre usuários
- Melhor organização do código (funções e modularização)
- Persistência de dados (salvar usuários em arquivo)

---

## 🛠️ Tecnologias utilizadas
- **Python 3**
- Biblioteca padrão (`time.sleep` para simulação de saída)
